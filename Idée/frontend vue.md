# Description Détaillée du Frontend – Projet Hafrose

## Présentation Générale

Le frontend de Hafrose sera développé avec React JS et Tailwind CSS afin d'offrir une expérience utilisateur moderne, élégante et fluide.

L'interface sera conçue autour d'une identité visuelle luxueuse mettant en valeur les accessoires de mode proposés par la marque.

L'objectif est de créer un site vitrine premium qui inspire confiance, reflète la qualité des produits et facilite la navigation des visiteurs sur tous les appareils.

---

# Identité Visuelle

## Style Graphique

Le design adoptera un style :

* Moderne
* Élégant
* Minimaliste
* Haut de gamme
* Professionnel

L'interface doit transmettre une image de luxe tout en restant simple et agréable à utiliser.

# Cartes Produits (Product Cards)

Chaque produit sera affiché dans une carte uniforme.

Contenu :

* Image du produit
* Nom
* Prix
* Catégorie
* Bouton de consultation

Caractéristiques :

* Taille identique pour toutes les cartes
* Coins arrondis
* Ombres modernes
* Effet Hover
* Bonne lisibilité

Exemple :

```text
┌─────────────────┐
│     Image       │
│                 │
├─────────────────┤
│ Nom Produit     │
│ Prix            │
│ Catégorie       │
│ [Voir Produit]  │
└─────────────────┘
```
